# Домашнее задание
## Общая задача
### Исходные данные

- Дано множество $A$ из $N$ натуральных чисел:  
  $A = \{x_1, x_2, x_3, \dots, x_n\}$
- Каждый элемент $x_i$ имеет **цену**, равную его индексу $i$.
- Изначально множество $B$ пусто.

### Цель

Переместить **все элементы** из множества $A$ в множество $B$, минимизируя **суммарную стоимость** операций.

### Разрешённые действия

За одну операцию разрешается:

1. Выбрать **два элемента** из текущего множества $A$ и переместить их в $B$.
2. Затем **вернуть один** из этих двух элементов обратно в $A$.
3. Стоимость операции равна **цене большего** (по индексу) из двух выбранных элементов.

### Примечания

- Элементы, однажды попавшие в $B$, могут в дальнейшем снова участвовать в операциях (если их вернули в $A$).
- Цена элемента фиксирована и не меняется со временем.
- После завершения всех операций:
  - $A$ должно быть пустым,
  - $B$ должно содержать все исходные элементы $\{x_1, x_2, \dots, x_n\}$.

### Требуется

Предложить алгоритм решения данной задачи и проанализировать его сложность.


## Предложенные алгоритмы
### Алгоритм №1
#### Вариант для $n$ - чётное

1. $A {x_1, x_2\} \to B$
2. $A \gets B {x_1\}$
3. $A {x_n, x_{n-1}\} \to B$
4. $n - 2$
5. $A \gets B {x_2\}$
6. _Повторять шаги_ $1 - 5$ _до тех пор пока в множестве A не останется_ ${x_1, x_2, x_3}$.
7. $A {x_1, x_2\} \to B$

#### Вариант для $n$ - нечётное

1. $A {x_1, x_2\} \to B$
2. $A \gets B {x_1\}$
3. $A {x_n, x_{n-1}\} \to B$
4. $n - 2$
5. $A \gets B {x_2\}$
6. _Повторять шаги_ $1 - 5$ _до тех пор пока в множестве A не останется_ ${x_1, x_2, x_3}$.
7.  $A {x_1, x_3\} \to B$
8.  $A \gets B {x_1\}$
9.  $A {x_1, x_2\} \to B$

### Алгоритм №2

10. $A {x_1, x_2\} \to B$

### Алгоритм №3
1 Этап
  * 1/3 от множества A переносим в множество B, по следующему алгоритму:
  * N = объем множества A
  1. $i = 1$
  2. $A {x_i, x_{i + 1}\} \to B$
  3. $A \gets B {x_1\}$
  4. $i + 2$
  5. $A {x_i, x_{i+1}\} \to B$
  6. $A \gets B {x_2\}$
  7. _Повторять шаги_ $2 - 6$ _до тех пор пока i не будет равна 1/3 от N_
  8. $A {x_1, x_2} \to B$
  9. $A \gets B {x_1\}$
     
2 Этап
  * Меняем 1/3 элементов находящихся в множестве B на 2/3 элементов находящихся в множестве A
   1. $i = 2 j = 1$
   2. $A {x_j, x_{j + 1}\} \to B$
   3. $A \gets B {x_i\}$
   4. $i + 1$
   5. $j + 2$
   6. _Повторять шаги_ $2 - 5$ _до тех пор пока i не будет равна 1/3 от N, а j  не будет равна 2/3 от N_
      
3 Этап
  * Оставшиеся элементы из множества A переносим в множество B
  1. $A {x_1, x_2\} \to B$
  2. $A \gets B {x_1\}$
  3. $A {x_n, x_{n-1}\} \to B$
  4. $n - 2$
  5. $A \gets B {x_2\}$
  6. _Повторять шаги_ $1 - 5$ _до тех пор пока задача не будет выполнена_

### Код программы для 1 алгоритма

```python
def algorithm_1(set_A: list, set_B: list) -> int:
    set_A.sort()
    set_B.sort()
    money_box = 0
    
    if len(set_A) % 2 == 0:

        while  len(set_A) != 2:
            # имитация перехода x_1 и x_2 в множество В    
            money_box += 2
            # имитация перехода x_1 обратно в множество А
            money_box += 1
            # переход x_n и x_(n-1) в множество B
            money_box += set_A[-1]
            set_B.extend([set_A.pop(-1), set_A.pop(-1)])
            # имитация перехода x_2 обратно в множество А
            money_box += 2

            set_B.sort()

        else:
            money_box += max(set_A)
            set_B.extend(set_A)
            set_A.clear()
    else:
        while  len(set_A) != 3:
            # имитация перехода x_1 и x_2 в множество В    
            money_box += 2
            # имитация перехода x_1 обратно в множество А
            money_box += 1
            # переход x_n и x_(n-1) в множество B
            money_box += set_A[-1]
            set_B.extend([set_A.pop(-1), set_A.pop(-1)])
            # имитация перехода x_2 обратно в множество А
            money_box += 2

            set_B.sort()

        else:
            set_A.sort()
            # имитация перехода x_1 и x_3 в множество B
            money_box += 3
            # имитация перехода x_1 обратно в множество A
            money_box += 1
            # имитация перехода x_1 и x_2 в множество В 
            money_box += 2

            # фактический переход всех оставшихся элементов в множество B
            set_B.extend(set_A)
            set_A.clear()


    set_B.sort()
    return money_box
```
### Код программы для 2 алгоритма

```python
def algorithm_2(set_A: list, set_B: list) -> int:
    set_A.sort()
    set_B.sort()

    money_box = 0

    # имитация перемещения x_1 и x_2 из множества A  в множество B
    money_box += 2
    # имитация перемещения x_1 обратно в множество B
    money_box += 1
    # перемещение x_n и x_(n-1) из множества A  в множество B
    money_box += set_A[-1]
    set_B.extend([set_A.pop(-1), set_A.pop(-1)])
    # имитация перемещения x_2 обратно в множество B
    money_box += 2
    while len(set_A) != 2:
        # перемещение x_(n-k) и имитация перемещения x_1 из множества A 
        # в множество B. А так же имитация возвращени x_1 обратно в множество B
        money_box += set_A[-1] + 1
        set_B.append(set_A.pop(-1))
    else:
        # перемещения x_1 и x_2 из множества A  в множество B
        money_box += max(set_A)
        set_B.extend(set_A)
        set_A.clear()
    set_B.sort()
    return money_box
```
