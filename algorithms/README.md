# Домашнее задание
## Общая задача
### Исходные данные

- Дано множество $A$ из $N$ натуральных чисел:  
  $A = \{x_1, x_2, x_3, \dots, x_n\}$
- Каждый элемент $x_i$ имеет **цену**, равную его индексу $i$.
- Изначально множество $B$ пусто.

### Цель

Переместить **все элементы** из множества $A$ в множество $B$, минимизируя **суммарную стоимость** операций.

### Разрешённые действия

За одну операцию разрешается:

1. Выбрать **два элемента** из текущего множества $A$ и переместить их в $B$.
2. Затем **вернуть один** из этих двух элементов обратно в $A$.
3. Стоимость операции равна **цене большего** (по индексу) из двух выбранных элементов.

### Примечания

- Элементы, однажды попавшие в $B$, могут в дальнейшем снова участвовать в операциях (если их вернули в $A$).
- Цена элемента фиксирована и не меняется со временем.
- После завершения всех операций:
  - $A$ должно быть пустым,
  - $B$ должно содержать все исходные элементы $\{x_1, x_2, \dots, x_n\}$.

### Требуется

Предложить алгоритм решения данной задачи и проанализировать его сложность.
