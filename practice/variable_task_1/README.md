# Аннотированный список литературы по направлению 09.03.01 «Информатика и вычислительная техника»

---

## 1. Компьютерные сети

**1. Олифер, В. Г.** Компьютерные сети. Принципы, технологии, протоколы : учебное пособие / В. Г. Олифер, Н. А. Олифер. — Юбилейное изд., доп. — Санкт-Петербург : Питер, 2026. — 1008 с. — (Учебник для вузов. Питер). — ISBN 978-5-4461-4085-5.

*Аннотация:*  
Юбилейное издание классического труда по сетевым технологиям. Около трети материала обновлено или написано заново: добавлен раздел о беспроводных сетях, переработаны главы по SDH, OTN, DWDM. Рекомендовано Минобрнауки РФ для направления «Информатика и вычислительная техника». Предназначено для студентов, аспирантов и технических специалистов.

**2. Таненбаум, Э. С.** Компьютерные сети : 6-е изд. / Э. С. Таненбаум, Д. Узеролл, Н. Фимстер ; перевод с англ. С. Черникова. — Санкт-Петербург : Питер, 2026. — 992 с. — (Классика Computer Science). — ISBN 978-5-4461-1766-6.

*Аннотация:*  
Полностью переработанное шестое издание самой авторитетной книги по сетевым технологиям. Рассмотрены DOCSIS, 4G/5G, 802.11ax, 100-гигабитный Ethernet, интернет вещей, QUIC, BBR, программно-конфигурируемые сети. Системно изложены уровни организации сетей (от физического до прикладного). Большое внимание уделено сетевой безопасности.

**3. Молочков, В. П.** Компьютерные сети. Создание и настройка в Cisco Packet Tracer : практикум / В. П. Молочков. — Санкт-Петербург : Наука и техника, 2025. — 432 с. — ISBN 978-5-907592-75-9.

*Аннотация:*  
Практико-ориентированное руководство по работе с эмулятором Cisco Packet Tracer. Содержит пошаговые примеры настройки сетей, задания для самостоятельной работы, вопросы для самопроверки. Материал изложен доступно, без требований к предварительной подготовке. Прилагаются готовые схемы (*.pkt) и экзаменационные билеты.

**4. Компьютерные сети для начинающих [Электронный ресурс] / sodacatit // Хабр. — 2025. — 6 июля. — Режим доступа: https://habr.com/ru/articles/925448/ (свободный). — Загл. с экрана.**

*Аннотация:*  
Вводная статья для начинающих IT-специалистов. Простыми словами с использованием аналогий объясняются IP-адресация, маски подсети, различия TCP/UDP, функции коммутаторов и маршрутизаторов, модели OSI и TCP/IP, технология NAT, основы IPv6. Материал помогает сформировать базовое понимание сетевых технологий.

**5. Бархатова, Д. А.** Компьютерные сети : учебник для вузов / Д. А. Бархатова, Д. Н. Буторин, А. А. Левин, Н. И. Пак, А. М. Панова. — Санкт-Петербург : Лань, 2025. — 304 с. — ISBN 978-5-507-51752-7. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/460616 (для авториз. пользователей).

*Аннотация:*  
Учебник предназначен для освоения темы «Компьютерные сети» учащимися 4–11 классов, а также студентами педагогических вузов естественно-научного цикла (математика, информатика, физика, биология и др.). Может быть использован в бакалавриате и магистратуре.

**6. Трошин, А. В.** Компьютерные сети : учебное пособие / А. В. Трошин. — Самара : ПГУТИ, 2024. — 184 с. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/463667 (для авториз. пользователей). — Дата обращения: 08.05.2026.

*Аннотация:*  
Учебное пособие разработано в соответствии с ФГОС ВО по направлению подготовки 11.03.02 «Инфокоммуникационные технологии и системы связи». Предназначено для студентов 3 курса очной формы обучения.

**7. Урбанович, П. П.** Компьютерные сети : учебное пособие / П. П. Урбанович, Д. М. Романенко. — Вологда : Инфра-Инженерия, 2022. — 460 с. — ISBN 978-5-9729-0962-9. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/281867 (для авториз. пользователей).

*Аннотация:*  
Даны общие понятия и определения из области компьютерных сетей и технологий, описаны сетевые компоненты, топологии, методы доступа к каналу связи, физические среды передачи данных. Рассмотрены эталонная модель OSI, типы сетевого оборудования, сетевые операционные системы. Для студентов технических специальностей высших учебных заведений.

## 2. Программирование

**8. Альтман, Е. А.** Система контроля версий GIT : учебно-методическое пособие к выполнению лабораторных работ / Е. А. Альтман, А. В. Александров, Т. В. Васеева. — Омск : ОмГУПС, 2021. — 26 с. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/190155 (для авториз. пользователей). — Дата обращения: 08.05.2026.

*Аннотация:*  
В пособии рассматриваются основные принципы работы с системой контроля версий Git: от описания базовых команд настройки и работы с репозиториями до организации работы с использованием клиент-серверной архитектуры. Для студентов, изучающих дисциплины «Информационные технологии», «Программирование», «Управление разработкой информационных систем».

**9. Анто, Т.** Сверхбыстрый Python : руководство / Т. Анто ; перевод с англ. А. Ю. Гинько. — Москва : ДМК Пресс, 2023. — 370 с. — ISBN 978-5-93700-226-6. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/456590 (для авториз. пользователей).

*Аннотация:*  
Книга посвящена написанию быстрого кода на чистом Python с оптимизацией на основе библиотек и аппаратного обеспечения. Рассмотрены алгоритмы и инструменты для анализа больших данных, параллельная обработка данных. Содержит примеры программного кода, адаптируемые к конкретным задачам.

**10. Давыдова, Н. А.** Программирование : учебное пособие / Н. А. Давыдова, Е. В. Боровская. — 5-е изд. (эл.). — Москва : Лаборатория знаний, 2025. — 241 с. — ISBN 978-5-93208-831-9. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/451607 (для авториз. пользователей). — Дата обращения: 08.05.2026.

*Аннотация:*  
Описаны теоретические положения и принципы разработки программ, критерии качества программ и этапы проектирования объектно-ориентированного программного продукта. Рассмотрены основные понятия языков программирования, основы программирования в Object Pascal, принципы визуального программирования, работа с базами данных и динамическими структурами.

**11. Истратова, Е. Е.** Алгоритмы и структуры данных : учебное пособие / Е. Е. Истратова, И. Н. Томилов, А. А. Якименко. — Новосибирск : НГТУ, 2025. — 68 с. — ISBN 978-5-7782-5502-9. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/514566 (для авториз. пользователей). — Дата обращения: 08.05.2026.

*Аннотация:*  
Кратко изложены базовые понятия теории алгоритмов, методы оценки временной и пространственной сложности алгоритмов, рассмотрены основные структуры данных, описаны способы задания графов и программных объектов. Рекомендовано для направления 09.03.01 «Информатика и вычислительная техника».

**12. Алгоритмы и структуры данных [Электронный ресурс] / che1nov // Хабр. — 2025. — 21 янв. — Режим доступа: https://habr.com/ru/articles/875200/ (свободный). — Загл. с экрана.**

*Аннотация:*  
Статья-роадмэп по изучению алгоритмов и структур данных. Рассмотрены базовые структуры (массивы, связные списки, деревья, хеш-таблицы, графы), алгоритмы поиска и сортировки, техники работы с указателями, обход графов (DFS, BFS). Содержит рекомендации по практике на платформах LeetCode и CodeWars.

**13. Основы Python за 1 статью: от Hello World до рабочего скрипта [Электронный ресурс] / enamored_poc // Хабр. — 2026. — 26 янв. — Режим доступа: https://habr.com/ru/articles/988966/ (свободный). — Загл. с экрана.**

*Аннотация:*  
Концентрат синтаксиса Python в процедурном стиле (без ООП). Рассмотрены переменные, типы данных, условные операторы, циклы, списки, словари, функции, работа с файлами, обработка ошибок, модули. Включён практический пример — игра «Угадай число». Для начинающих программистов.

**14. Шукарёв, И. А.** Основы языка Python : учебное пособие / И. А. Шукарёв. — Ульяновск : УлГТУ, 2025. — 126 с. — ISBN 978-5-9795-2533-4. — Текст : электронный // Лань : электронно-библиотечная система. — Режим доступа: https://e.lanbook.com/book/518800 (для авториз. пользователей). — Дата обращения: 08.05.2026.

*Аннотация:*  
В учебном пособии изложены основы синтаксиса, ключевые возможности и особенности работы со структурами данных в языке Python. Предназначено для студентов бакалавриата направления 09.03.02 «Информационные системы и технологии» (смежное с 09.03.01).
